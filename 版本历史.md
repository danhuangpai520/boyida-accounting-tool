## v1.1 - 2026-06-03

虚拟机和无 NVIDIA 显卡场景的本地 OCR 修正版。

### 交付内容

- 主程序：`做账执行工具.exe`
- 版本号：`1.1`
- 文件大小：`11,920,880` 字节
- SHA256：`7E11EE67C2076A9BB506F65ED05B9D7013565FDEC3B1CFE0B7C50924E32F507F`
- 本地备份：`项目资料\版本备份\20260603_020501_新版`

### 本版修复

- 本地 PaddleOCR 只把 NVIDIA 显卡作为 GPU 候选；虚拟机显卡、核显、Microsoft Basic Render Driver 不再误装 GPU 包。
- 无 NVIDIA 或虚拟机环境选择本地 OCR 时，自动安装/使用 CPU 版 PaddlePaddle，并明确提示本次调用 `cpu`。
- Paddle 环境检测改为无缓冲输出，并延长验证时间，避免首次导入或 `run_check()` 慢时误判“已安装但未通过”。
- CPU 模式下不再尝试 TensorRT / FP16 / HPI 参数，避免把 GPU 加速参数套到 CPU 环境。
- 本地 OCR 安装命令加入 `--prefer-binary` 和更长 pip 超时，降低下载依赖卡住的概率。
- 打包脚本可在没有 `builtin_api_key.txt` 时使用本机已保存 Key 注入 EXE，全程不打印 Key。

- 版本号：`1.0`

### 当前能力

- 支持一键导入运输磅单图片、OCR 识别、结构化解析并生成 Excel。
- 默认使用在线智谱 GLM-OCR，也保留本地 PaddleOCR PP-OCRv5 方案入口。

- 支持 GPS 授权配置、启动刷新和右上角面板手动刷新。

- GitHub：进入 Releases，下载 `v1.0` 的 `做账执行工具.exe`。
